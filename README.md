## Prerequisits

* node 8+

## Quickstart

### Set up accounts with aws / now

```
npm run init
```

### Deploy code to now and spec to AWS

```shell
npm run deploy
```

## Test

```shell
npm test
```

## Test manually in AWS

```shell
npm run ask:testing
```
